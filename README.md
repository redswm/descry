# descry
JS скрипты для слабовидящих

Для работы нужно встроить скрипты через плагин, например, User JavaScript and CSS V 3.1.2 для Google Chrome

<h2>Список возможностей</h2>

<ul>
  <li>Numpad
    <ul>
      <li>Numpad- Открывает Уведомления</li>
      <li>Numpad+ Открывает Лиды</li>
      <li>Numpad* Читает вслух верхнее уведомление в Уведомлениях</li>
    </ul>
  </li>  
  <li>CSS
    <ul>
      <li>Вместо исчезающей кнопки закрытия уведомлений увеличинная кнопка "XXX"</li>      
    </ul>
  </li>
  <li>Функции
    <ul>
      <li>flashScreen() запускает мерцание экрана</li>      
    </ul>
  </li>
</ul>
