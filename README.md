# descry
JS скрипты для слабовидящих

Для работы нужно встроить скрипты и css в браузер через плагин, например, User JavaScript and CSS для Google Chrome

<h2>Список возможностей</h2>

<ul>
  <li>Numpad
    <ul>
      <li>Numpad- Открывает Уведомления</li>
      <li>Numpad+ Открывает Лиды</li>
      <li>Numpad* Читает вслух верхнее уведомление в Уведомлениях | Последнее сообщение в мессенджере | Заголовок страницы</li>
      <li>Numpad* Если вызвано на странице сделки, копирует email в буфер</li>
      <li>Стрелка Вверх - Читает предыдущее</li>
      <li>Стрелка Вниз - Читает следующее</li>
    </ul>
  </li>  
  <li>CSS
    <ul>
      <li>Вместо исчезающей кнопки закрытия уведомлений увеличинная кнопка "XXX"</li>      
    </ul>
  </li>
  <li>Функции
    <ul>
      <li>flashScreen() запускает мерцание экрана</li>      
       <li>playSuccessSound() запускает короткий звуковой сигнал "Успешно"</li>      
      
    </ul>
  </li>
</ul>
