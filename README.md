# descry
JS скрипты для слабовидящих

Для работы нужно встроить скрипты через плагин, например, User JavaScript and CSS V 3.1.2 для Google Chrome

Список возможностей:<br><br>

Numpad- Открывает уведомления<br>
Numpad+  Открывает Лиды<br>
Numpad*  Читает вслух верхнее уведомление в Уведомлениях<br><br>

Вместо кнопки закрытия уведомления, которая появляется только при наведении показывается "XXX"

Функция flashScreen () запускает мерцание на экране
